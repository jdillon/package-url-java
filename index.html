Welcome

Implementation of Package URL specification for Java.

Dependency

To use from Apache Maven include dependency:

<dependencyManagement>
  <dependencies>
    <dependency>
      <groupId>org.sonatype.goodies</groupId>
      <artifactId>package-url-java</artifactId>
      <version>1.0.0</version>
    </dependency>
  </dependencies>
</dependencyManagement>
<dependency>
  <groupId>org.sonatype.goodies</groupId>
  <artifactId>package-url-java</artifactId>
  <version>1.0.0</version>
</dependency>
